## About the Notes
This repository contains concise, example-driven notes that cover the essential building blocks of JavaScript. It’s designed as a quick reference for beginners and a handy refresher for anyone preparing for interviews or building projects.

**Purpose**  
- Capture key concepts from Sheriyans Coding School lectures  
- Provide runnable examples and short explanations  
- Serve as a personal cheat sheet for future review

---

## Topics Covered
These notes focus on the foundational topics every JavaScript developer should master:

- **Variables and Declarations** — `var`, `let`, `const`, hoisting, best practices  
- **Data Types and Type System** — primitives, objects, type coercion, `typeof`  
- **Operators** — arithmetic, comparison, logical, assignment, ternary, nullish coalescing  
- **Control Flow** — `if`, `else`, `switch`, short-circuit patterns  
- **Loops** — `for`, `while`, `do while`, `for...of`, `for...in`, iteration patterns  
- **Functions** — declarations, expressions, arrow functions, default params, rest and spread  
- **Arrays and Objects** — creation, methods, iteration, destructuring, immutability patterns

Each topic includes short code snippets, common pitfalls, and quick examples you can run in the browser console.
